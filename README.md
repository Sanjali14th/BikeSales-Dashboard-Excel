# Bike Sales Dashboard (Excel)

## Project Overview
This project focuses on analyzing bike sales customer data using Microsoft Excel to understand purchasing behavior across different customer segments. The work involved cleaning and validating raw data, performing exploratory analysis using pivot tables, and building an interactive dashboard for business insights.

## Key Tasks Performed
- Cleaned and structured raw customer sales data in Excel
- Validated data and standardized categorical fields for consistency
- Created pivot tables to analyze bike purchases across age groups, gender, income levels, education, region, and commute distance
- Built an interactive Excel dashboard using charts and slicers for dynamic analysis
- Enabled quick comparison of purchase behavior across multiple demographic dimensions

- Higher income levels are associated with increased likelihood of bike purchases
