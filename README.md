# Bike Sales Dashboard (Excel)

## Project Overview
This project involves analyzing bike sales customer data using Microsoft Excel. The objective was to clean and structure raw data, perform exploratory analysis using pivot tables, and build an interactive dashboard to understand customer purchasing behavior.

## Key Activities
- Cleaned and structured raw bike sales data
- Standardized categorical values and handled inconsistencies
- Created pivot tables to analyze sales trends across demographics
- Analyzed purchase behavior based on age, gender, income, marital status, region, and commute distance
- Designed an interactive Excel dashboard for quick insights and reporting

## Insights Covered
- Relationship between income level and bike purchases
- Impact of commute distance on buying decisions
- Bike purchase trends across age groups and regions
- Gender-wise and marital status-based analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Dashboard Elements
